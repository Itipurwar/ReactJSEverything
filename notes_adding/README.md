# Notes Adding App

A simple React app to **add, view, and delete notes**.  
Notes are saved in **Local Storage** so they persist on page reload.

## Features
- Add new notes
- View all saved notes
- Delete individual notes
- Notes persist using Local Storage

## Technologies Used
- React.js
- HTML, CSS, JavaScript

## How to Run
1. Clone the repository  
   ```bash
   git clone <your-repo-link>

2. Navigate to the project folder
cd notes_adding

3. Install dependencies
npm install

4. Start the app
npm run dev
npm start
